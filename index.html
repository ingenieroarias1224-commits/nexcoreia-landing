FAQ

Preguntas frecuentes

¿Qué es NexcoreIA SAS?
+

NexcoreIA SAS es una empresa colombiana especializada en automatización empresarial y desarrollo de software con inteligencia artificial. Nuestro producto estrella es el asistente de causaciones automáticas para contadores, pero también desarrollamos software a medida y automatizamos procesos para empresas de todos los sectores.

¿Por qué son únicos con OCR en Colombia?
+

Desarrollamos tecnología propia de OCR específicamente para facturas electrónicas DIAN de Colombia (UBL 2.1). Mientras otros softwares requieren digitación manual, nuestro sistema extrae automáticamente todos los datos usando inteligencia artificial Claude de Anthropic. No existe otro software colombiano con esta capacidad.

¿Con qué ERPs se integra?
+

Nos integramos nativamente con SIESA Enterprise, Siigo Nube y Helisa — los 3 ERP más robustos del mercado contable colombiano. La causación aprobada se envía automáticamente via API sin doble digitación. También desarrollamos integraciones con otros sistemas según sus necesidades.

¿Desarrollan software personalizado?
+

Sí. Además del producto de causaciones, NexcoreIA SAS desarrolla software a medida: portales web, sistemas internos, integraciones entre plataformas, automatización de procesos y apps. Contáctenos para una consulta sin costo sobre su proceso.

¿Cómo funciona la captura desde el correo?
+

Configura el correo IMAP (Gmail, Outlook) una sola vez. El sistema revisa automáticamente la bandeja, descarga los XML adjuntos y los causa sin que el contador haga nada. También descomprime ZIPs con XMLs adentro.

¿Cuánto tiempo toma implementar?
+

Menos de 20 minutos. Configure PUC, credenciales SIESA y correo IMAP. A partir de ahí el sistema trabaja solo. No requiere instalación ni conocimientos técnicos. Nuestro equipo le acompaña en la configuración inicial.
